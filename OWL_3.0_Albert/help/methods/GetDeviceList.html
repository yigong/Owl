GetDeviceList
Syntax
CameraAPI.GetDeviceList( list );
Parameters
String[] list - (optional) Used to override the list of available devices (drivers) as read from the operating system. Can be used to allow Owl 2.0 to operate with a version 1.7 driver by setting list to {astropci0, etc}.
Returns
String[] - The list of available devices (drivers) or null if no pci boards (drivers) exist. If a board is in the system and null is returned, then the correct driver may not be installed or the board may require servicing.
Remarks
Retrieves the list of available pci devices (drivers). Throws a java.lang.Exception if an error occurs, which can be caught using a try/catch block.
