WriteFitsComment
Syntax
CameraAPI.WriteFitsComment( comment, filename );
Parameters
String comment A comment tag to add to the FITS header.
String filename The filename of the FITS file to modify.
Returns
None
Remarks
Write a comment tag to the specified FITS file. Uses the COMMENT keyword. Throws java.lang.Exception if an error occurs. All exceptions can be caught using a try/catch block.
