IsCCParamSupported
Syntax
CameraAPI.IsCCParamSupported( parameter );
Parameters
int parameter The parameter to check support for. See the list of controller parameters for details.
Returns
boolean - True if the specified parameter is supported by the controller; false otherwise.
Remarks
Verifies that the specified controller configuration parameter is supported. Throws java.lang.Exception if an error occurs. All exceptions can be caught using a try/catch block.
