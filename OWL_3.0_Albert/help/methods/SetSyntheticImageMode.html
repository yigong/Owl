SetSyntheticImageMode
Syntax
CameraAPI.SetSyntheticImageMode( mode );
Parameters
boolean mode True to activate synthetic image mode; false otherwise.
Returns
None
Remarks
Sets the camera image mode to synthetic, which produces an image that continuously ramps up over the 16-bit data range from 0 to 65535 adu. The image is produced by the timing board DSP and not the camera array. Throws java.lang.Exception if an error occurs. All exceptions can be caught using a try/catch block.
