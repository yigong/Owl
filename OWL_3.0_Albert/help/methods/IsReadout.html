IsReadout
Syntax
CameraAPI.IsReadout();
Parameters
None
Returns
boolean - True if the controller is currently in readout; false otherwise.
Remarks
Returns true if the image is currently in readout; false otherwise. All commands, except CameraAPI.ABORT_READOUT, all blocked by the pci board and will not be forwarded to the controller. Throws java.lang.Exception if an error occurs. All exceptions can be caught using a try/catch block.
