SubtractImageHalves
Syntax
CameraAPI.SubtractImageHalves( rows, cols );
Parameters
int rows The row size of the image.
int cols The column size of the image.
Returns
None
Remarks
Causes the image buffer to subtract one half from the other. The row and column parameters specify the total image size and not the size of each half. Throws java.lang.Exception if an error occurs. All exceptions can be caught using a try/catch block.
