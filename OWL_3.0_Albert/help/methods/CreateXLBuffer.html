CreateXLBuffer
Syntax
CameraAPI.CreateXLBuffer( rows, cols );
Parameters
int rows The row size of the image buffer.
int cols The column size of the image buffer.
Returns
None
Remarks
Creates an extra-large image buffer that is made-up of a chain of smaller 2048x2048 buffers. Owl uses these methods when the image to readout exceeds 4200x4200 pixels. Throws java.lang.Exception if an error occurs. All exceptions can be caught using a try/catch block.
