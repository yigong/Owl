GetPCIStatus
Syntax
CameraAPI.GetPCIStatus();
Parameters
None
Returns
int - The pci status value: 0 (timeout), 5 (readout), and 6 (busy).
Remarks
Returns the current value of the pci boards HTF (Host Transfer Flags) bits (3,4 & 5) from the HSTR register. Throws java.lang.Exception if an error occurs. All exceptions can be caught using a try/catch block.
