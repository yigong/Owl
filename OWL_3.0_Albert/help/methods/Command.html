Command
Syntax
CameraAPI.Command( board_id, command, arg1, arg2, arg3, arg4 );
Parameters
int board_id Board to send the command to: CameraAPI.PCI_ID (pci board), CameraAPI.TIM_ID (timing board), or CameraAPI.UTIL_ID (utility board).
int command A valid pci, timing, or utility board command. See command list for details.
Returns
int - A command specific reply value. For most commands the reply will be CameraAPI.DON (done).
Remarks
Sends a command to the pci, timing, or utility board. Throws a java.lang.Exception if an error occurs, which can be caught using a try/catch block.
