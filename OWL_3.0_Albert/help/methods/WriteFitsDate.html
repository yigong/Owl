WriteFitsDate
Syntax
CameraAPI.WriteFitsDate( filename );
Parameters
String filename The filename of the FITS file to modify.
Returns
None
Remarks
Writes the current date and time to the specified FITS file. Uses the DATE keyword and writes the data in the following format: 'yyyy-mm-ddThh:mm:ss'. Throws java.lang.Exception if an error occurs. All exceptions can be caught using a try/catch block.
