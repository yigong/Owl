Exception
Syntax
java.lang.Exception
Type
class
Remarks
The standard java exception class that is the base for most other exceptions. You can catch most exceptions by catching this exception with a try/catch block.

Example:
If any error occurs with the method call, an Exception will be thrown. 
	try
	{
		... do something here, such as ...

		CameraAPI.WriteFitsFile( "Image.fit", 600, 512 );
	}
	catch ( Exception e }
	{
		... do something here,
                such as log an error ...

		logger.error( e );
	}
