DateTimeUtility
Syntax
DateTimeUtility();
Parameters
None
Returns
DateTimeUtility Object

Class Methods
METHOD DESCRIPTION
String getDateString() Returns the date in the following format: month/day/year. Example: 4/6/2009.
String getTimeString() Returns the time in the following format: hour:minute:second:millisecond. Example: 17:6:41:654

Remarks
A utility for getting the current date and time.

Example: 
	dateTime = DateTimeUtility();
	System.out.println( "Date: " + dateTime.getDateString() );
	System.out.println( "Time: " + dateTime.getTimeString() );
