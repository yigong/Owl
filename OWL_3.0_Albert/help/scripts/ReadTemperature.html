ReadTemperature
Syntax
ReadTemperature( demoMode );
Parameters
boolean demoMode True to run in demo mode; false otherwise. Demo mode allows testing of the ARC_API CController temperature code by using a valid adu value (internal). Setting to false will read the temperature value from the controller
Returns
None; runs in graphical user interface mode only.

Class Methods
None

Remarks
Reads the camera temperature using a graphical user interface (GUI). This script cannot be run without a GUI. It can be run in demo mode for purposes of testing the underlying API code.

Example: 
	ReadTemperature( false );	// read controller temperature (non-demo mode)

	ReadTemperature( true );	// read controller temperature (demo mode)
