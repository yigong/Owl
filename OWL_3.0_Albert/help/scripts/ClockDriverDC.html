ClockDriverDC

ClockDriverDC();
Parameters
None

javax.swing.JFrame

Class Methods
This class executes in a graphical user interface when instantiated, so no methods can be called externally.

Remarks
Creates a graphical user interface for setting the clock driver parameters. This is for the system developer and is not something the standard user should attempt to use.
