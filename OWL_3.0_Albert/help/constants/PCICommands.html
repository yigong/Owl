PCI Board Commands
Syntax
CameraAPI.RESET_CONTROLLER
CameraAPI.RESET_PCI
CameraAPI.ABORT_READOUT
Type
int
Remarks
Integer constants that represent specific commands to the pci board. See list below.

REPLYDESCRIPTION
RESET_CONTROLLERresets the controller; power will be turned off
RESET_PCIresets the pci board by resetting the DSP program counter
ABORT_READOUTtells the pci board and controller to stop readout
